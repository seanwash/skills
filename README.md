# Skills

Custom [Agent Skills](https://platform.claude.com/docs/en/agents-and-tools/agent-skills/overview) for Claude.

## Usage

Copy skill directories to your Claude Code skills location:
- Personal: `~/.claude/skills/`
- Project: `.claude/skills/`

## Use with Codex

Copy skill directories into your Codex skills location:
- Default: `~/.codex/skills/`
- Custom: `$CODEX_HOME/skills/`

Then reference the skill from your `AGENTS.md` (e.g., `skill-installer`) or by name in a prompt.
